The Dot Factory is a GPL license program that can be used to
convert TTF fonts for .c files.  The latest Windows binary
and source files can be found at:

http://www.pavius.net/downloads/tools/53-the-dot-factory

The binary files are included here purely for
convenience sake.

The following fonts are used in the LPC1343
Code Base:

DejaVu - http://dejavu-fonts.org/wiki/Main_Page
Bitsream Vera - http://www.gnome.org/fonts/
